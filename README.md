## 简介
一款第三方网易云音乐下载软件

## 声明
本软件仅供学习交流、UI测试使用，下载后请务必于 24 小时内删除

## 已支持的特性
- 搜索/下载音乐
- 解析歌单
- 导入歌曲元数据
- 下载歌词
- 登录账号（填入cookie、验证码）

## 待实现的特性
- 邮箱登录
- 自定义主题色
- 自定义歌曲文件命名规则
